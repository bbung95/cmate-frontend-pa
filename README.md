# C-MATE frontend PA

C-MATE Frontend PA 를 구현하였습니다.

## 프로젝트 실행

패키지 매니져로 pnpm을 사용하였습니다.

```bash
# clone
git clone https://github.com/bbung95/cmate-frontend-pa.git

# change directory
cd cmate-frontend-pa

# install
pnpm install

# start
pnpm dev
```

## Figma MCP

figma custom mcp를 사용하여 디자인 토큰을 구현하였습니다.
