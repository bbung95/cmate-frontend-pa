# frontend PA

Frontend PA 를 구현하였습니다.

## 프로젝트 실행

패키지 매니져로 pnpm을 사용하였습니다.

```bash
# clone
git clone https://github.com/bbung95/frontend-pa.git

# change directory
cd frontend-pa

# install
npm install
# or pnpm install

# start
npm run dev
# or pnpm dev
```

## Figma MCP

figma custom mcp를 사용하여 디자인 토큰과 디자인컴포넌트를 구현했습니다.
